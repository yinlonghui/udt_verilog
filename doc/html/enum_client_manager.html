详细描述

本模块实例化了listen,ProcessClose,close模块，主要功能管理SERVER端SOCKET套接字，监视其状态

            ClientManager功能如下：
            1、向Server端发送请求，通过3次握手后，连接建立完成
            2、处理用户关闭连接
            3、处理对端关闭包
